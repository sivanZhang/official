Register into system

          Go to the register page:Click here to register
          Input your email address(Please use the arcserve email account), which is used to receive the random password, and system will automatically check the uniqueness of your email address. This password is automatically generated by system. After log in, you can change the password by clicking the 'Change Password' link on the top right corner.
          Then input your first name and last name, click 'Register' to finish this operation.
          You will get your password in your email box. And you will be add to L1 group by default in this system, if you want more permissions, please contact the administrators.
          After register, Click here to log in.

What can I do in this system?

      1 Find a Problem

          In the host page, click 'Problem', then the problems which have been approved by approvers will be listed in this page.
          Search Problem:
          By now, user can search problems according to the Problem title, the first name of Creator and the keywords.
          If you want list all the Problem records with your permissions, just clear the three text fields and click 'Search'.
           Obviously this search feature is not the final version, we will upgrade it according to users' feedback.
          The detail of a Problem:
          This is quite easy, just click the Problem title you will find the detail of a Problem and the Fixes which are created based on the current Problem. If you click the Fix title and find the prompt message is 'permission denied' that means the Fix is not available to you now.

      2 Find a Fix

          There are two ways to find a Fix, the first way is that you want to find the Fix based on a certain Problem. In this situation you need to find the Problem first and then click the Problem title to the Problem detail page to find the related Fixes just like what we described in the first section 'Find a Problem'.
          The second way, in the host page, click 'Fix', then the Fixes which have been approved by approvers will be listed in this page.
          Search Fix:
          By now, user can search Fixes according to the Fix title, the first name of Creator and the keywords.
          If you want list all the Fix records with your permissions, just clear the three text fields and click 'Search'.
           Obviously this search feature is not the final version, we will upgrade it according to users' feedback.
          The detail of a Fix:
          Just like the Problem, click the Fix title you will find the detail of the Fix and the Problem which is used to create the Fix. You can download the binary files of the Fix.

      3 Download the binary files of a Fix

          When you find a Fix and go to the detail page of a Fix, you can download the binary files of a Fix.

      4 Add Zendesk Ticket No. to a Fix

          When you find a Fix and go to the detail page of a Fix, you can add the Ticket # to Fix. First input the Ticket #, and click 'Add to List' to add the Ticket # to List, at last click 'Save Changes' to save.
           Remember if 'Save Changes' not be clicked, no Ticket # will be added to the Fix after you leave or refresh current page.
